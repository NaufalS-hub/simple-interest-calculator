# Simple Interest Calculator

A simple shell script that calculates simple interest given principal, annual rate of interest and time period in years.

## Description:
This project is designed as part of the Final Project for the Git and GitHub course. The main goal is to demonstrate the ability to manage repositories, handle documentation files (README, License, Code of Conduct), and implement basic Bash scripting.

## Input:
* `p`: Principal amount
* `t`: Time period in years
* `r`: Annual rate of interest

## Output:
* `Simple Interest = p * t * r`

## How to run:
1. Clone this repository.
2. Ensure the script has execution permissions: `chmod +x simple-interest.sh`
3. Run the script: `./simple-interest.sh`
